Membuat Aplikasi ATM Sederhana Menggunakan Java Berbasis CLI

Tutorial ini akan mengajarkan langkah demi langkah membangun simulasi sistem perbankan dasar menggunakan konsep Switch-Case, Loop, dan Input Scanner pada Java.

Pendahuluan

Salah satu cara terbaik untuk mempelajari logika pemrograman dasar adalah dengan meniru sistem dunia nyata. Dalam studi kasus ini, kita akan membuat ATM Console Application. Proyek ini mencakup penggunaan variabel, logika percabangan (if-else / switch), dan perulangan (do-while).

Persiapan Tools

Sebelum memulai, pastikan Anda telah menginstal:

  • JDK (Java Development Kit): Versi 8 atau terbaru.
  • Code Editor: VS Code, IntelliJ IDEA, atau NetBeans.

Kode Program Lengkap

Berikut adalah implementasi penuh dari file ATM.java. Kode ini menangani input user untuk cek saldo, setor tunai, dan tarik tunai.

ATM.java 
package ATM_Sederhana;

import java.util.Scanner;

public class ATM {
    public static void main(String[] args) {
        // Inisialisasi saldo awal
        int saldo = 1000000;
        int pilihan;
        Scanner input = new Scanner(System.in);

        do {
            System.out.println("\n=== ATM SEDERHANA ===");
            System.out.println("1. Cek Saldo");
            System.out.println("2. Tarik Tunai");
            System.out.println("3. Setor Tunai");
            System.out.println("4. Keluar");
            System.out.print("Pilih transaksi: ");

            pilihan = input.nextInt();

            switch (pilihan) {
                case 1:
                    System.out.println("Saldo Anda: Rp " + saldo);
                    break;
                case 2:
                    System.out.print("Nominal tarik: Rp ");
                    int tarik = input.nextInt();
                    if (tarik > saldo) {
                        System.out.println("Saldo tidak cukup!");
                    } else {
                        saldo -= tarik;
                        System.out.println("Berhasil! Sisa saldo: Rp " + saldo);
                    }
                    break;
                // ... logic case lainnya
                case 4:
                    System.out.println("Terima kasih!");
                    break;
                default:
                    System.out.println("Input tidak valid.");
            }
        } while (pilihan != 4);
        
        input.close();
    }
}

Penjelasan Logika

Mari kita bedah bagian-bagian penting dari kode di atas:

1. Scanner Input

Kita menggunakan java.util.Scanner untuk menangkap input dari keyboard pengguna, baik itu untuk memilih menu maupun memasukkan nominal uang.

2. Do-While Loop

Perulangan do-while memastikan menu ATM terus muncul kembali setelah transaksi selesai, sampai pengguna memilih opsi Keluar (4).

Kesimpulan

Proyek sederhana ini dapat dijadikan sebagai latihan dasar untuk pemula dalam memahami konsep dasar pemrograman Java. Dengan memahami konsep dasar ini, Anda dapat membangun aplikasi yang lebih kompleks dan berbasis CLI.
