Membangun Aplikasi ATM Sederhana dengan Java

Eksplorasi logika pemrograman fundamental menggunakan Java untuk menciptakan sistem manajemen transaksi yang efisien dan mudah digunakan.

Fungsionalitas Sistem ATM (Berdasarkan Kode Sumber)

Program ini diimplementasikan menggunakan kelas ATM dalam paket ATM_Sederhana. Sistem menggunakan variabel integer saldo dengan nilai awal sebesar Rp 1.000.000 sebagai basis data transaksi. Berikut adalah rincian logika transaksinya:

01

Cek Saldo (Check Balance)

Sistem memanggil nilai variabel saldo saat ini dan menampilkannya langsung ke konsol. Ini memberikan informasi instan mengenai dana yang tersedia sebelum melakukan transaksi lain.

02

Tarik Tunai (Withdrawal)

Logika ini melibatkan pengecekan kondisi if-else. Jika jumlah penarikan lebih besar dari saldo, sistem menolak transaksi dengan pesan "Saldo tidak mencukupi". Jika valid, saldo akan dikurangi secara otomatis.

03

Setor Tunai (Deposit)

Fitur penambahan saldo di mana input dari pengguna ditambahkan langsung ke variabel saldo utama melalui operasi aritmatika penambahan. Saldo terbaru akan ditampilkan segera setelah setoran berhasil.

04

Keluar (Exit System)

Sistem akan menghentikan perulangan do-while dan menampilkan pesan "Terima kasih telah menggunakan ATM kami" sebagai penutup sesi transaksi.

Mekanisme Kontrol & Validasi

Seluruh interaksi dibungkus dalam blok perulangan do-while yang terus berjalan selama pilihan pengguna bukan nomor 4. Selain itu, terdapat blok default pada struktur switch untuk menangani input angka yang tidak tersedia di menu, sehingga program tidak langsung berhenti saat terjadi kesalahan input.
