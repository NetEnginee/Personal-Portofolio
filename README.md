## ğŸš€ Features

Website ini dilengkapi dengan berbagai fitur modern untuk memastikan pengalaman pengguna terbaik:

- **Responsive Design:** Tampilan fleksibel untuk Desktop, Tablet, dan Mobile.
- **Dark Mode UI:** Menggunakan skema warna `#121212` dengan aksen Emas (`#d4af37`) untuk kesan elegan.
- **SEO Optimized:** Dilengkapi dengan Meta Tags, Schema.org JSON-LD, dan Sitemap.
- **Blog/Knowledge Hub:** Kumpulan artikel teknis seputar Networking, SysAdmin, dan Programming dengan sistem filtering kategori.
- **Interactive Elements:**
  - Animasi Scroll Reveal.
  - Typing Effect pada Hero Section.
  - Download CV dengan Toast Notification.
  - Certificate Lightbox (Popup) untuk melihat detail sertifikat tanpa pindah halaman.
  - Preloader Animation untuk loading state yang smooth.
  - Dynamic Category Filtering di halaman blog.
